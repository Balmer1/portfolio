Proyecto de investigación

Agustín Balmer

Robótica y Fenotipado de Alta Capacidad con Relevamiento de Datos en Campo
Aplicaciones en Agricultura de Precisión

Descripción

Diseño y desarrollo de una plataforma de censado y seguimiento para relevamiento de datos a campo y manejo de grandes volúmenes de datos.

Tecnologías

  • Raspberry pi 3
  • Raspbian Wheezy
  • Ubuntu 16.04.1 LTS
  • Oracle VM VirtualBox
  • Python 3.0
  • OpenCV 3.1
  • TKInter
  • Tetracam adc lite

Descripción

El sistema desarrollado se encarga analizar un stream de video de una cámara de inferior calidad, reconociendo plantas de maíz y realizando un tracking de estas para recolectar una serie de imágenes de una cámara de superior calidad para su posterior análisis. El hardware del sistema es una plataforma Raspberry Pi 3 con sistema operativo Raspbian y pantalla táctil integrada, una cámara web genérica (video), una cámara multiespectral tetracam (imagen) y dos baterías portátiles. El software consiste de un sistema desarrollado en python utilizando las librerías OpenCV para la detección/tracking y Tkinter para la interfaz gráfica.

Equipo de trabajo

Este es un trabajo conjunto entre la Secretaría de Investigación, Desarrollo y Transferencia de la UNNOBA y el INTA, Estación Experimental Pergamino. El proyecto se desarrolla en el Instituto de Investigación en Tecnologías y Transferencia (ITT) dependiente de la mencionada Secretaría, y se trabaja en conjunto con la Escuela de Tecnología de la UNNOBA.
